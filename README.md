MongoDB Restaurants Queries
This file contains basic MongoDB queries for the "Restaurants" collection.

It includes:
* Find data → find()
* Select fields → projections (choose specific fields)
* Filter data → by borough, cuisine, score, etc.
* Conditions → $gt, $lt, $ne, $in, $nin
* Logical operators → $and, $or
* Regex → search by name patterns
* Sorting → ascending / descending
* Pagination → limit() and skip()
* Field checks → $exists, $type, $mod

Purpose → These queries show how to retrieve and filter restaurant data using different MongoDB features.
